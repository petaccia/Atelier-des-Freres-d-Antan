# Atelier Frères d'Antan - Frontend

Ce dossier contient le frontend de l'application Atelier Frères d'Antan, développé avec Next.js.

## Technologies utilisées

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Swiper

## Installation

```bash
cd frontend
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement

Le déploiement est géré par Vercel.

```bash
vercel
```

Pour déployer en production :

```bash
vercel --prod
```
